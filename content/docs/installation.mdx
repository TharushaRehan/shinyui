## Next.js

Follow the steps to use Shiny UI components in a Next.js project.

### Installation

<Steps>

<Step>To create a Next.js project, run the following command.</Step>

```bash
npx create-next-app@latest my-app
```

<Step>After creating the project, navigate to the project directory.</Step>

```bash
cd my-app
```

<Step>Install Framer Motion.</Step>

Shiny UI uses Framer Motion for animations. To install Framer Motion, run the following command:

```bash
npm install framer-motion
```

</Steps>
