## Props

| Prop        | Type     | Default | Description                                      |
| ----------- | -------- | ------- | ------------------------------------------------ |
| className   | string   |         | The class name to apply to the component.        |
| text        | string   | false   | The text should appear with the cursor.          |
| cursorColor | string   |         | Background color of the cursor.                  |
| translateX  | [string] |         | Defines the horizontal movement range in pixels. |
| translateY  | [string] |         | Defines the vertical movement range in pixels.   |
